# 🚀 快速开始指南

> 5 分钟了解整个系统的工作原理

## 🎯 这是什么？

这是一个**自动发推文机器人**，可以：
- ✅ 定时从 Google Sheet 读取文章
- ✅ 自动发布到 Twitter
- ✅ 支持文字和图片
- ✅ 完全免费（使用 GitHub Actions）

## 🔄 工作流程图

```
┌─────────────────┐
│ 你在 Google     │
│ Sheet 填写      │ ← 1. 你只需要做这一步
│ 文章信息        │
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│ GitHub Actions  │
│ 定时检查        │ ← 2. 自动运行
│ （每天2次）     │
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│ 发现新文章      │
│ → 格式化推文    │ ← 3. 自动处理
│ → 下载图片      │
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│ 发布到 Twitter  │ ← 4. 自动发布
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│ 更新 Google     │
│ Sheet 状态      │ ← 5. 自动更新
│ 为"已发送"      │
└─────────────────┘
```

## 📝 你需要做什么？

### 一次性配置（约 1 小时）

1. **申请 Twitter 开发者账号**（15 分钟）
   - 访问 https://developer.twitter.com
   - 填写申请表格
   - 等待审核

2. **创建 Google Cloud 项目**（15 分钟）
   - 创建服务账号
   - 下载 JSON 密钥

3. **准备 Google Sheet**（5 分钟）
   - 创建表格
   - 设置列标题
   - 分享给服务账号

4. **配置 GitHub**（10 分钟）
   - 创建仓库
   - 添加 Secrets

### 日常使用（每天 5 分钟）

**你只需要做一件事：在 Google Sheet 添加文章**

| 标题 | DOI链接 | TOC图片 | 发送状态 | 发送时间 | 推文ID |
|------|---------|---------|----------|----------|--------|
| 你的文章标题 | https://doi.org/xxx | | 未发送 | | |

就是这样！系统会自动处理剩下的所有事情。

## 🆘 遇到问题？

### 常见问题

1. **Twitter API 申请被拒绝**
   - 重新申请，提供更详细的使用说明
   - 说明是个人学习项目

2. **Google Cloud 配置复杂**
   - 按照教程一步一步操作
   - 使用配置检查清单验证每一步

3. **GitHub Actions 失败**
   - 查看 Actions 日志
   - 检查 Secrets 是否正确配置

### 获取帮助

1. 查看 [配置检查清单](CONFIGURATION_CHECKLIST.md)
2. 查看 Actions 运行日志
3. 查看 [常见问题解答](FAQ.md)

## 🎉 下一步

配置成功后，你可以：

1. **批量添加文章**

   - 修改 GitHub Actions 的 cron 表达式

4. **准备精美图片**
   - 上传到 `assets` 目录
   - 或使用图片 URL
